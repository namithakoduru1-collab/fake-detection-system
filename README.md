# Face Detection System

This project is a simple face detection application built using Python and OpenCV. It uses a webcam to capture live video and detects human faces in real time using a pre-trained Haar Cascade classifier. Detected faces are highlighted with a rectangle on the screen.

## Features

* Real-time face detection using webcam
* Detects multiple faces simultaneously
* Highlights detected faces with bounding boxes

## Technologies Used

* Python
* OpenCV

## Project Structure

face-detection-system
│
├── face_detection.py
├── requirements.txt
└── README.md

## Installation

Install the required library:

pip install opencv-python

## How to Run

Run the following command:

python face_detection.py

The webcam will open and the system will start detecting faces. Press **q** to close the application.
